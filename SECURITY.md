# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

If you find a security issue in PromptStrike itself (not in the targets it scans), please report it responsibly:

1. **Do not open a public issue.**
2. Email: v3n0msh3ll@protonmail.com
3. Include steps to reproduce and potential impact.
4. Allow 48 hours for an initial response.

## Scope

This policy covers the PromptStrike framework code only. Vulnerabilities found in third-party AI providers using this tool should be reported to those providers directly.
